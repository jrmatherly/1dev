# 1Code Documentation

Canonical documentation site for the 1Code enterprise fork (apollosai.dev).

## Local Development

```bash
cd docs
bun install --frozen-lockfile
bunx xyd          # dev server at http://localhost:5175
bun run build     # static output to .xyd/build/client/
```
